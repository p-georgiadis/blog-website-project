About This Blog

This website is a project created by Panagiotis Georgiadis as part of the Frontend Web Development Nanodegree provided by Udacity. The course was fully sponsored through the OneTen scholarship that I was awarded.

In this blog, I explore the futuristic world of Cyberpunk, from cybernetic enhancements to virtual realities. This project showcases advanced frontend web development skills, utilizing HTML, CSS (including Grid and Flexbox), and design principles that bring this vision to life.
